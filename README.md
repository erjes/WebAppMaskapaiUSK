# WebAppMaskapaiUSK

## Cara Menjalankan

Ikuti langkah-langkah berikut:

1. Clone repository:

    ```bash
    git clone https://github.com/erjes/WebAppMaskapaiUSK
    ```

2. Rename `.env.example` Menjadi `.env`.

3. Install Composer:
   ```bash
    composer install
    ```

5. Jalankan migration:

    ```bash
    php artisan migrate
    ```

6. Start Server:

    ```bash
    php artisan serve
    ```
